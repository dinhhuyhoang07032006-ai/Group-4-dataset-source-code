# Project: Predicting Project Budget Overruns Using Machine Learning Techniques

## Project Overview
This project aims to predict project budget overruns by applying machine learning techniques within the Orange Data Mining environment. The study investigates key factors contributing to budget deviations and evaluates the effectiveness of predictive models in assisting project managers with cost control.

## Key Features
- **Data Preprocessing:** Cleaning and preparing cost accounting data to ensure model accuracy.
- **Predictive Modeling:** Utilizing machine learning algorithms (such as classification/regression) to forecast budget performance.
- **Analysis:** Evaluating model performance and identifying critical features that influence budget overruns.

## Repository Structure
- `cost_accounting.ows`: The main Orange Data Mining workflow file.
- `Dynamic_Cost_Accounting_Dataset.csv`: The raw dataset used for analysis.
- `Dynamic_Cost_Accounting_Dataset_clean.xlsx`: The processed dataset after data cleaning.
- `Dynamic_Cost_Accounting_Dataset_prediction.xlsx`: Results and predictions generated by the models.
